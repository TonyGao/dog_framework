# 测试账号清单

用于系统测试的预置账号及密码。

## 1. 三员管理账号 (System Officers)

适用于系统配置、安全管理及审计。

| 角色           | 用户名      | 密码          | 职责                         |
| :------------- | :---------- | :------------ | :--------------------------- |
| **系统管理员** | `sys_admin` | `Officer@123` | 系统软硬件环境配置、网络维护 |
| **安全保密员** | `sec_admin` | `Officer@123` | 用户账号管理、权限分配       |
| **安全审计员** | `auditor`   | `Officer@123` | 查看日志、审计操作           |

> **注意**: 登录时请直接输入**用户名**。

| 账号类型     | 邮箱 / 用户名 / 工号 (登录账号) | 密码          | 说明           |
| :----------- | :------------------------------ | :------------ | :------------- |
| **普通员工** | `emp_1@example.com`             | `password123` | 使用邮箱登录   |
| **普通员工** | `emp_1`                         | `password123` | 使用用户名登录 |
| **普通员工** | `EMP_1`                         | `password123` | 使用工号登录   |
| ...          | ...                             | ...           | ...            |

> **注意**: 员工登录可以使用**邮箱地址**、**用户名**或**工号**。

## 3. 其他测试账号 (OrgUser)

适用于外部用户或非员工体系用户（来自 Fixtures 数据）。

| 账号类型     | 用户名       | 密码          | 说明                                     |
| :----------- | :----------- | :------------ | :--------------------------------------- |
| **外部用户** | _(随机生成)_ | `password123` | 需查看数据库 `org_user` 表确认具体用户名 |

---

**提示**: 如需重置三员账号，请运行:

```bash
php bin/console app:init-officers
```

如需重新加载员工测试数据，请运行:

```bash
php bin/console ef:fixtures:load --all
```
