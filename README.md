## 狗狗框架

![狗狗框架图片](/documents/conceptions/assets/dog215.png)

### 功能概述

* 组织架构
  * 公司架构
  * 部门管理
  * 岗位管理
  * 职级管理
* 人员管理
  * 人员信息
* 模型和接口
  * 模型管理
  * 接口管理
* 应用程序构建器
  * 表单设计器
  * [菜单管理](/documents/features/Menu.md)
